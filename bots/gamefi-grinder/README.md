# 🎮 GameFi 掛機 Bot

自動玩 Web3 Game 賺 Token（Play-to-Earn 自動化）

## 策略
- 自動 login wallet → 自動點擊/種田/打怪
- 自動 claim reward 到固定時間
- 多 account 輪換（避免被 ban）
- Discord 通知每日收入

## 例子遊戲
| 遊戲 | 做咩 | 賺咩 |
|------|------|------|
| Axie Infinity | 自動打 PvE | SLP token |
| Pixels | 自動種田/收成 | BERRY token |
| 各類 Telegram Game | auto-tap | Token airdrop |

## 成本 vs 回報
| 項目 | 數值 |
|------|------|
| 啟動成本 | 通常 $0（free-to-play） |
| 每日收入 | $1-$10+ |
| 每月收入 | $30-300+ |
| 風險 | 低（game token 可能跌價） |

## ⚠️ 真實情況
2026 年 GameFi 熱度比以前跌咗好多。大部分 Telegram tap-to-earn game 嘅 token 上完幣就歸零。真正賺到錢嘅 game 得幾個頭部項目。
